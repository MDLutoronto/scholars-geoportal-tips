# Helpful tips when using Scholars GeoPortal

Please [visit this link](http://guides.scholarsportal.info/geoportal/usinggeoportal) for extensive help with Scholars GeoPortal.

[Scholars GeoPortal](https://go.openathens.net/redirector/utoronto.ca?url=http://geo2.scholarsportal.info) is a geospatial discovery tool that provides access to large scale geospatial datasets that can be used for mapping or analysis. These datasets encompass both vector and raster data on a variety of topics such as land-use, transportation networks, census geography, aerial imagery, geology, and more. Furthermore, users can customize their data extracts.

**Tools:** [Scholars GeoPortal](https://mdl.library.utoronto.ca/tools/scholars-geoportal-0)
